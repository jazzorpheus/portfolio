Projects

Weather Map (Early version!)

A front-end weather app created in React that utilizes the OpenWeatherMap, Tomorrow.io, and Mapbox API's to render live weather data together with an integrated map for displaying current weather patterns such as precipitation intensity and wind speed. Only in its early stages, the plan is to continue improving the UI with better graphics/visualizations, and eventually add full forecasting in the form of dashboards, plus a time slider on the map that can extend backwards or forwards in time.

Stock Trader App

A web app created with the Flask framework together with a SQL database where users can 'purchase' and 'sell' stocks based on real-time data pulled from the IEX Exchange via the IEX Cloud API. This project was part of an assignment for HarvardX's CS50 course. See markdown on GitHub for more details.

React Shopping List

A shopping/groceries list web app made with React and Material UI that allows the user to create, check off, and delete shopping list items grouped by category. The user can optionally input product prices and quantites which the app keeps track of and renders total costs.

Spellchecker

A command-line application written in C that enables users to spellcheck .txt files. The output includes a list of misspelled words plus stats on the time taken to execute various parts of the program. The primary purpose of the project was to investigate appropriate data structures for loading the dictionary into memory, e.g. hash tables.
